**Article 1er.** A l'annexe de l'arrêté du Gouvernement de la Communauté française du 19 juillet 2007 reconnaissant et admettant aux subventions les formations organisées par la Haute Ecole Mons-Borinage-Centre à partir de l'année académique 2007-2008, sont apportées les modifications suivantes :

   a) la ligne :

```
  Type court   Economique     Specialisation "Management de    Mons
                               la distribution"
```



est insérée entre les lignes suivantes :

```
  Type court   Economique     Section "Tourisme" - Option      Saint-Ghislain
                               "Gestion"
  Type court   Paramedicale   Section "Accoucheuse"            Mons
```



   b) la ligne :

```
  Type court   Paramedicale   Section "Accoucheuse"            Mons
```



est remplacée par la ligne

```
  Type court   Paramedicale   Section "Sage-femme"             Mons
```



**Art. 2.** Le présent arrêté entre en vigueur à partir de l'année académique 2008-2009.

**Art. 3.** Le Ministre ayant l'Enseignement supérieur dans ses attributions est chargé de l'exécution du présent arrêté.

Bruxelles, le 18 juillet 2008.

Pour le Gouvernement de la Communauté française :

La Vice-Présidente et Ministre de l'Enseignement supérieur, de la Recherche scientifique et des Relations internationales,

Mme M.-D. SIMONET.
