**Article 1er.** A l'annexe de l'arrêté du Gouvernement de la Communauté française du 19 juillet 2007 reconnaissant et admettant aux subventions les formations organisées par la Haute Ecole provinciale de Charleroi-Université du Travail à partir de l'année académique 2007-2008, sont apportées les modifications suivantes :

   a) la ligne :

```
  Type court   Technique     Section "Biotechnique" - Finalite      Charleroi
                              "Bioelectronique et instrumentation"
```



est insérée entre les lignes

```
  Type long    Paramedical   Section "Kinesitherapie" 2e cycle      Charleroi
  Type court   Technique     Section "Informatique et systemes" -     Tournai
                              Finalite "Automatique"
```



   b) les lignes

```
  Type long    Technique   Section "Sciences industrielles" -       Charleroi
                            Finalite "Chimie"
  Type long    Technique   Section "Sciences industrielles" -       Charleroi
                            Finalite "Informatique"
```



sont supprimées;

   c) la ligne

```
  Type long    Technique   Section "Sciences industrielles" -       Tournai
                            Finalite "Electromecanique"
```



est remplacée par la ligne

```
  Type long    Technique   Section " Sciences industrielles" -        Tournai
                            Finalite "Industrie"
```



**Art. 2.** Le présent arrêté entre en vigueur à partir de l'année académique 2008-2009.

**Art. 3.** Le Ministre ayant l'Enseignement supérieur dans ses attributions est chargé de l'exécution du présent arrêté.

Bruxelles, le 18 juillet 2008.

Pour le Gouvernement de la Communauté française :

La Vice-Présidente et Ministre de l'Enseignement supérieur, de la Recherche scientifique et des Relations internationales,

Mme M.-D. SIMONET.
