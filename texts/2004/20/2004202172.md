**Article 1er.** A l'article 1er de l'arrêté du Gouvernement de la Communauté française du 23 décembre 2003 fixant les échelles de traitement du personnel administratif, du personnel spécialisé, du personnel de maîtrise, des gens de métier et de service des universités et faculté universitaire de la Communauté française, sont apportées les modifications suivantes :

   a) Dans la catégorie 1. " Catégorie du personnel de direction et attaché ", le point 5. est remplacé par les mots

```
  " 5. Premier conseiller           (14/1)  29.669,24   14(2)  x  1.324,48";
```



   b) Dans la catégorie 2. " Catégorie du personnel administratif, adjoint à la recherche, de gestion, puéricultrice, surveillant de travaux, et dessinateur ", le point 3. est remplacé par les mots :

```
  " 3. Agent spécialisé             (20/2x)  13.557,50    3(1) x  259,82
                                                          2(2) x  250,45
                                                          6(2) x  606,08
                                                          1(2) x  608,58
                                                          4(2) x  618,08";
```



   c) Dans la catégorie 3. " Personnel paramédical ", le point 1. est remplacé par les mots :

```
  " 1. Infirmier gradue             (22/6)   15.460,16    3(1) x  303,01
                                                          4(2) x  519,49
                                                          1(2) x  523,75
                                                          7(2) x  529,80 ";
```



   d) Dans la catégorie 3. " Personnel paramédical ", le point 7. est remplacé par les mots :

```
  " 7. Logopède de 1re classe       (23/6)  17.451,49     3(1) x  303,01
                                                          1(2) x  522,02
                                                         11(2) x  529,80 ";
```



   e) Dans la catégorie 3. " Personnel paramédical ", le point 8. est remplacé par les mots :

```
  " 8. Logopède principal           (24/6) 19.250,50      3(1) x  308,98
                                                         12(2) x  706,33 ";
```



   f) Dans la catégorie 3. " Personnel paramédical ", le point 10. est remplacé par les mots :

```
  " 10. Logopède en chef principal (25/6) 20.486,77       3(1) x  529,80
                                                         12(2) x  706,33 ";
```



   g) Dans la catégorie 4. " Catégorie du personnel spécialisé ", le point 11. est remplacé par les mots :

```
  " 11. Ingénieur industriel        (10/1) 20.398,35      3(1) x  618,08
                                                         10(2) x  949,21

                                                         10(2) x  949,21 ";
```



   h) Dans la catégorie 4. " Catégorie du personnel spécialisé ", le point 33. est remplacé par les mots :

```
  " 33. Informaticien               (12/1) 25.254,58      3(1) x  618,08
                                                         11(2) x  949,21
  après cinq ans                    (12/2) 31.214,44      3(1) x  662,20
                                                          8(2) x 1324,48 ".
```



**Art. 2.** Le présent arrêté produit ses effets le 1er janvier 2003.

**Art. 3.** Le Ministre de la Fonction publique et la Ministre de l'Enseignement supérieur et de la Recherche scientifique sont chargés de l'exécution du présent arrêté.
