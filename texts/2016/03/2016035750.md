**Article 1er.** A l'article 1er de l'arrêté ministériel du 20 juillet 2015 relatif aux zones de médecins généralistes, sont apportées les modifications suivantes :

1. dans le point 1°,  le membre de phrase " , Affligem " est inséré ;

2. le point 2°  est abrogé ;

3. dans le point 3°,  le membre de phrase " , Ranst, Schilde " est abrogé ;

4. il est ajouté un point 86°,  rédigé comme suit : " 86°  Voorkempen : Schilde, Wijnegem ; "

**Art. 2.** Le présent arrêté produit ses effets le 1er janvier 2016.
