**Article 1er.** Sont désignés en qualité de service hospitalier universitaire, pour le nombre de lits mentionnés, les services suivants :

```
  1. A l'hopital civil de Charleroi :
  Chirurgie                                 index C :              20 lits
  Medecine interne                          index D :              25 lits
                                                        Total :    45 lits
  2. A l'hopital Andre Vesale a Montigny-le-Tilleul :
  Medecine interne                          index D :              55 lits
  3. Au Centre hospitalier universitaire de Tivoli a La Louviere :
  Chirurgie                                 index C :              25 lits
  Medecine interne                          index D :              25 lits
                                                        Total :    50 lits
  4. A l'hopital Ambroise Pare a Mons :
  Pediatrie                                 index E :              20 lits
  5. A l'Institut medico-chirurgical a Tournai :
  Medecine interne                          index D :              15 lits
  6. A l'hopital de la Madeleine a Ath :
  Chirurgie                                 index C :              15 lits
```



**Art. 2.** Le présent arrêté produit ses effets le 5 février 1996.

**Art. 3.** Notre Ministre des Affaires sociales est chargé de l'exécution du présent arrêté.

Donné à Bruxelles, le 4 mars 1998.

ALBERT

Par le Roi :
