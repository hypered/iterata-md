**Article 1er.** Le nombre d'huissiers de justice est fixé comme suit par arrondissement judiciaire :

_Table:_

| Arrondissement judiciaire | Nombre |
| --- | --- |
| Anvers | 80 |
| Limbourg | 29 |
| Bruxelles | 87 |
| Louvain | 17 |
| Brabant wallon | 14 |
| Flandre orientale | 58 |
| Flandre occidentale | 49 |
| Eupen | 2 |
| Liège | 62 |
| Luxembrourg | 17 |
| Namur | 28 |
| Hainaut | 66 |


**Art. 2.** L'arrêté royal du 26 avril 2009 fixant le nombre d'huissiers de justice par arrondissement judiciaire est abrogé.

**Art. 3.** Le ministre qui a la Justice dans ses attributions est chargé de l'exécution du présent arrêté.
