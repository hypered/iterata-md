**Article 1er.** Dans l'annexe III de l'AR/CIR 92, remplacée par l'arrêté royal du 9 décembre 2021, les modifications suivantes sont apportées :

1. dans le tableau b) du numéro 2.2., A., les mots "ou un parent divorcé ou séparé de fait," sont remplacés par les mots "ou un parent divorcé, séparé de fait ou séparé de corps," ;

2. dans le texte néerlandais du tableau a) du numéro 2.10., A, les mots "21.795,00 EUR" sont remplacés par les mots "22.290,00 EUR" ;

3. dans le texte néerlandais du numéro 4.3., B, les mots "141 euro" sont remplacés par les mots "144 euro" ;

4. dans les numéros 6.1., A., 1., 6.1., A., 2. et 6.1., B., 2., a) et b), les mots "1er janvier 2021" sont à chaque fois remplacés par les mots "1er janvier 2022" ;

5. dans le texte néerlandais du numéro 6.3., B., a), 1., les mots "1.710 euro" sont remplacés par les mots "1.749 euro".

**Art. 2.** Le présent arrêté est applicable aux revenus payés ou attribués à partir du 1er janvier 2022.

**Art. 3.** Le ministre qui a les Finances dans ses attributions, est chargé de l'exécution du présent arrêté.
