**Article 1er.** En application de l'article 111 du décret du 25 février 1997 relatif à l'enseignement fondamental, les établissements d'enseignement fondamental spécial suivants obtiennent une autorisation de création :

1. VBSBO Zonnehart (26419), Sportlaan 18, 8211 Aartrijke, pour le type 9 de l'enseignement maternel ;

2. VLSBO Zonnebloem (26666), Slabbaardstraat-Noord 90, 8870 Izegem, pour le type 9 de l'enseignement maternel et primaire ;

3. VLSBO Bernadetteschool Grotenberge (27102), Parkstraat 2, 9620 Zottegem, pour le type 9 de l'enseignement primaire.

**Art. 2.** En application des articles 286, 289 et 290/1 du Code de l'Enseignement secondaire du 17 décembre 2010, les établissements d'enseignement secondaire spécial suivants obtiennent une autorisation de programmation :

1. GO! SBSO Zonnegroen (46417), Sint-Truidensesteenweg 44, 3440 Zoutleeuw, pour les types 2 et 9 dans la forme d'enseignement 2 ;

2. BuSO Sint-Gerardus (27862), Sint-Gerardusdreef 1, 3590 Diepenbeek, pour le type 4 dans la forme d'enseignement 4 ;

3. Ziekenhuisschool Asster Sint-Truiden - Alexianen Tienen; Melveren-centrum 111, 3800 Sint-Truiden, pour le type 5 dans la forme d'enseignement 4 ;

4. BuSO Sint-Gregorius (28241), Jules Destréelaan 67, 9050 Gentbrugge, pour le type 3 dans la forme d'enseignement 4 ;

5. GO! SBSO De Varens (27284), Nieuwe Sint-Annadreef 27, 8200 Sint-Andries, pour les types 3 et 9 dans la forme d'enseignement 4 ;

6. Provinciale BUSO De Wissel, Arbeidstraat 66, 3600 Genk, pour le type 9 dans la forme d'enseignement 4 ;

7. VTS3 OV4B, Breedstraat 104, 9010 Sint-Niklaas, pour le type 9 dans la forme d'enseignement 4.

**Art. 3.** En application de l'article 287, § 3, du Code de l'Enseignement secondaire du 17 décembre 2010, l'établissement d'enseignement secondaire spécial suivant obtient une autorisation de création d'une implantation du type 5 : Ziekenhuisschool Asster Sint-Truiden - Alexianen Tienen; Melveren-centrum 111, 3800 Sint-Truiden, une implantation pour le type 5 dans la forme d'enseignement 4 à Psychiatrische kliniek Alexianen, Liefdestraat 10, 3300 Tienen.

**Art. 4.** Le présent arrêté entre en vigueur le 1er septembre 2019.

**Art. 5.** La Ministre flamande ayant l'enseignement dans ses attributions est chargée de l'exécution du présent arrêté.
