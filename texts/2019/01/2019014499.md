**Article 1er.** A l'annexe IIIbis de l'AR/CIR 92, insérée par l'arrêté royal du 22 août 2006, remplacée par l'arrêté royal du 31 juillet 2009 et modifiée par les arrêtés royaux des 21 février 2014, 28 avril 2015, 23 août 2015, 19 juillet 2018 et 13 avril 2019, les modifications suivantes sont apportées :

1. le code "44 heures supplémentaires (Art. 2751, alinéa 4, 2ème tiret, CIR 92) est remplacé par le code "44 heures supplémentaires (art. 2751, alinéa 4, deuxième tiret, et alinéa 7, CIR 92)" ;

2. le code "45 heures supplémentaires (Art. 2751, alinéa 4, 1er tiret, CIR 92)" est remplacé par le code "45 heures supplémentaires (art. 2751, alinéa 4, premier tiret, et alinéa 7, CIR 92)" ;

3. le code "51 heures supplémentaires (art. 2751, alinéa 4, 2e tiret, et art. 2751, alinéa 7, CIR 92)" est remplacé par le code "51 heures supplémentaires (art. 2751, alinéa 4, deuxième tiret, et alinéa 8, CIR 92)" ;

4. le code "52 heures supplémentaires (art. 2751, alinéa 4, 1er tiret, et art. 2751, alinéa 7, CIR 92)" est remplacé par le code "52 heures supplémentaires (art. 2751, alinéa 4, premier tiret, et alinéa 8, CIR 92)" ;

5. le code "55 Horeca (Art. 2751, alinéa 4, deuxième tiret, et alinéa 8, CIR 92)" est remplacé par le code "55 Horeca (art. 2751, alinéa 4, deuxième tiret, et alinéa 9, CIR 92)".

**Art. 2.** Le présent arrêté produit ses effets aux rémunérations payées ou attribuées à partir du 1er janvier 2019.

**Art. 3.** Le ministre qui a les Finances dans ses attributions est chargé de l'exécution du présent arrêté.
