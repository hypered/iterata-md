1. dans la première phrase le nombre " 140,04 " est remplacé par le nombre " 135,54 " ;

2. dans le tableau le nombre " 74,04 " est remplacé par le nombre " 70,04 " ;

3. dans le tableau le nombre " 8,5 " est remplacé par le nombre " 8 " ;

4. dans le tableau le nombre " 140,04 " est remplacé par le nombre " 135,54 ".
