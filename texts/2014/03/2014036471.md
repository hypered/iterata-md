**Article 1er.** Les pouvoirs organisateurs mentionnés ci-après sont autorisés à commencer leurs programmations approuvées dans leurs établissements d'enseignement artistique à temps partiel à partir de l'année scolaire 2014-2015 :

1. l'administration communale de Genk, pour la programmation de l' option " animatiefilm " dans le degré moyen de l'orientation " Arts plastiques " dans l' " Academie voor Beeldende Kunst en Media " de Genk ;

2. l'administration communale de Kortrijk, pour la programmation de l' option " steen-beeld " dans le degré de spécialisation de l'orientation " Arts plastiques " dans la " Koninklijke Academie voor Schone Kunsten " de Kortijk ;

3. l'administration communale de Hasselt, pour la programmation de l' option " toegepaste grafiek " dans le degré de spécialisation de l'orientation " Arts plastiques " dans la " Stedelijke Academie voor Schone Kunsten " de Hasselt ;

4. l'administration communale de Gent, pour la programmation de l' option " animatiefilm " dans le degré de spécialisation de l'orientation " Arts plastiques " dans la " Stedelijke Academie voor Beeldende Kunst " de Gent ;

5. l'administration communale de Zwevegem, pour la programmation de l' option " zang " dans le degré moyen de l'orientation " Musique " dans la " Kunstacademie " de Zwevegem ;

6. l'administration communale de Asse, pour la programmation de l' option " zang / jazz en lichte muziek " dans le degré supérieur de l'orientation " Musique " dans la " Kunstacademie " de Asse ;

7. l'administration communale de Herentals, pour la programmation de l'option " volksmuziek " dans le degré supérieur de l'orientation " Musique " dans la " Stedelijke Academie voor Muziek, Woord en Dans " de Herentals ;

8. le groupe d'écoles Zuid-Limburg, pour la programmation de l'option " toneel " dans le degré supérieur de l'orientation " Arts de la parole " dans l' " Academie van het GO! " de Tongeren ;

9. l' administration communale de Londerzeel, pour la programmation de l'option " dans en muziek " dans le degré supérieur de l'orientation " Danse " dans la " Gemeentelijke Academie " de Londerzeel.

**Art. 2.** Le Ministre flamand qui a l'enseignement dans ses attributions est chargé de l'exécution du présent arrêté.
