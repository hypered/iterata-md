**Article 1er.** Dans l'annexe de l'arrêté ministériel du 23 mars 2005 fixant les tarifs pour des prestations effectuées par la Bibliothèque royale de Belgique, modifié en dernier lieu par l'arrêté ministériel du 31 août 2015, la rubrique D est remplacée comme suit:

_Table:_

|  | D. Droits d'entrée à la Librairie des Ducs de Bourgogne (KBR Museum) |
| --- | --- |
| EUR | Droits d'entrée |
| 11,00 | 1. Prix plein (par personne) |
| 8,00 | 2. Prix réduit |
| 8,00 | - Groupe (min. 15 personnes) |
| 8,00 | - Adultes à partir de 65 ans |
| 8,00 | - Etudiants en possession d'une carte valable |
| 0,00 | 3. Gratuité |
| 0,00 | - Enfants et jeunes jusqu'à 18 ans inclus |
| 0,00 | - Personne souffrant d'un handicap et leur accompagnateur (1 accompagnateur maximum par personne handicapée) |
| 0,00 | - Personnel enseignant |
| 0,00 | - Carte BELSPO |
| 0,00 | - Presse (avec carte de presse valable) |




**Art. 2.** Le présent arrêté produit ses effets le 18 septembre 2020.
