**Article 1er.** Les référentiels de validation des compétences relatifs au métier de conducteur de chariot élévateur sont approuvés comme suit :

_Table:_

| Métier  | Titres  | Code  |
| --- | --- | --- |
| Conducteur de chariot élévateur (H/F)  | Unité de Compétence n° 1 : Vérifier et utiliser un matériel motorisé de type gerbeur | CHAR-1 |
|  | Unité de Compétence n°  2 : Vérifier et utiliser un matériel motorisé de type frontal | CHAR-2 |
|  | Unité de Compétence n°  2 : Vérifier et utiliser un matériel motorisé de type frontal | CHAR-3  |


**Art. 2.** Le présent arrêté entre en vigueur le 1er mars 2018.

Le Membre du Collège ayant la Formation professionnelle dans ses attributions est chargé de l'exécution du présent arrêté.
