**Article 1er.** En application des articles 101, 103 et 111 du décret du 25 février 1997 relatif à l'enseignement fondamental, les établissements d'enseignement fondamental spécial suivants obtiennent une autorisation de programmation :

1. Het Vlot (26765), Sint-Elisabethstraat 6, 8900 Ypres, pour la création d'un type 3 (maternel et primaire) ;

2. nouvelle école KBO type 3 Renaix (numéro d'établissement à attribuer), Eisdaele 1, 9600 Renaix, pour la création d'un type 3 (maternel et primaire) ;

3. GO! BSBO Woudlucht (3401), Prosperdreef 3, 3001 Heverlee, pour la création d'un type 7 (primaire), en tant que type de libre choix.

**Art. 2.** En application de l'article 111 du décret du 25 février 1997 relatif à l'enseignement fondamental, l'établissement d'enseignement fondamental spécial suivant obtient une autorisation conditionnelle de programmation : GIBLO Beerse (128579), Antwerpseweg 48, 2340 Beerse, pour la création d'un type 2 (maternel et primaire). La condition à remplir est la création par l'école d'une offre dans laquelle on se concentre sur l'ensemble du groupe cible du type 2 tel que visé à l'article 10, § 1er, 2° du même décret.

**Art. 3.** En application des articles 15, 286, 289 et 290/1 du Code de l'Enseignement secondaire du 17 décembre 2010, les établissements d'enseignement secondaire spécial suivants obtiennent une autorisation de programmation :

1. Buso Huize Tordale (28019), Bruggestraat 39, 8820 Torhout, pour le type 2 dans la forme d'enseignement 1 ;

2. Buso Hasp-O 7 (109942), Naamsesteenweg 167, 3800 Saint-Trond, pour le type 3 dans la forme d'enseignement 2 ;

3. Buso Ganspoel (137431), Avenue Georges Henri, 1200 Bruxelles, pour le type 2 dans la forme d'enseignement 1 ;

4. Buso Sint-Idesbald (28118), De Zilten 52, 8800 Roulers, pour une nouvelle forme d'enseignement 2, type 3 ;

5. Buso Sint-Idesbald (28118), De Zilten 52, 8800 Roulers, pour une nouvelle forme d'enseignement 2, type 9 ;

6. RHIZO 7 Buso De Lage Kouter (112789), Beekstraat 113 B, 8500 Courtrai, pour une nouvelle forme d'enseignement 4, type 9 avec une nouvelle implantation : Beheerstraat 10, 8500 Courtrai ;

7. nouvelle école Ennea (numéro d'établissement à attribuer), Koninklijke Baan 5, 8420 De Haan, pour la création d'une forme d'enseignement 1 et 4, type 9.

**Art. 4.** Le présent arrêté entre en vigueur le 1er septembre 2024.

**Art. 5.** Le ministre flamand qui a l'enseignement et la formation dans ses attributions est chargé de l'exécution du présent arrêté.
