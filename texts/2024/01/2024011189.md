**Article 1er.** Les référentiels de validation des compétences relatifs au métier de " Aide-ménager titres-services/Aide-ménagère titres-services " sont adoptés comme suit :

_Table:_

| Métier  | Titre  | Code  |
| --- | --- | --- |
| Aide-ménager titres-services/Aide-ménagère titres-services  | UC1 : Entretenir l'habitation  | AIMTS1  |
|  | UC2 : Entretenir le linge  | AIMTS2  |
|  | UC3 : Faire les courses ménagères  | AIMTS3  |
|  | UC4 : Préparer les aliments  | AIMTS4  |




**Art. 2.** Le présent arrêté entre en vigueur le jour de sa signature.
