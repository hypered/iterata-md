**Article 1er.** Article 1. Article unique. Les formations auxquelles s'applique la clause d'écolage sont les suivantes:

1. Les formations donnant accès à une certification de type " Master " ;

2. les formations suivantes dans le domaine " Prévention, bien-être et sécurité au travail " :

   a) Prévention incendie A1 ;

   b) Certificat interuniversitaire en Victimologie et Psychotraumatologie ;

   c) Examinateurs aéromédicaux - Formation de base ;

   d) Examinateurs aéromédicaux - Formation avancée ;

   e) Masterclass Bien-être au Travail ;

   f) " Postgraduaat " Conseiller en prévention Niveau A ;

   g) Conseiller en prévention Ergonomie ;

   h) Conseiller en prévention Niveau 1 ;

   i) Conseiller en prévention Niveau 1 - Spécialisation ;

   j) Conseiller en prévention Niveau 2 ;

   k) Conseiller en prévention Niveau 2 - Module multidisciplinaire de base ;

   l) Conseiller en prévention Aspects psychosociaux ;

   m) Conseiller en prévention Risques psychosociaux ;

   n) Certificat d'université de Conseiller en prévention spécialisé en hygiène du travail ;

   o) Certificat d'université de Conseiller en prévention Niveau 1 ;

   p) Coordinateur sécurité et santé sur les chantiers Niveau A ;

   q) Conseiller généraliste en accessibilité ;

   r) Technicien agréé combustible gazeux type GI ;

   s) " Disability Management " ;

3. les formations suivantes dans le domaine : " Cybersécurité, protection des données et IT service management " :

   a) FOR508 : Réponse avancée aux incidents, chasse aux menaces et criminalistique numérique ;

   b) FOR577 : Réponse aux incidents et chasse aux menaces sous Linux ;

   c) Délégué à la protection des données ;

   d) SANS SEC617: " Wireless Penetration Testing and Ethical Hacking " ;

   e) SEC401: " Security Essentials - Network, Endpoint, and Cloud " ;

   f) Contre-mesures de surveillance technique ;

4. les formations suivantes dans le domaine : " Données, statistiques et développement logiciel  " ;

   a) Programmation en " Statistical Analysis System " 1 ;

   b) Programmation en " Statistical Analysis System " 2 - Techniques de manipulation des données ;

   c) Programmation en " Statistical Analysis System " 3 - Techniques avancées ;

   d) Python - Formation de base ;

   e) Python - Formation avancée (modélisation, examen et validation des données) ;

   f) Python - Intermédiaire 1 (acquisition, préparation et exploration des données) ;

   g) Python - Intermédiaire 2 (analyse des données) ;

   h) " Statistical Analysis System Visual Analytics " 1 pour SAS Viya - Formation de base ;

   i) " Statistical Analysis System Visual Analytics " 2 - Formation avancée ;

   j) " Statistical Analysis System " Macro-langue 1 ;

   k) " Structured Query Language " - Formation de base ;

   l) " Structured Query Language " - Formation avancée ;

   m) Statistiques descriptives - Formation de base ;

   n) HarvardX - " Introduction to Probability " ;

   o) Statistiques - Formation avancée ;

   p) Construire des applications web avec Angular et TypeScript ;

5. les formations suivantes dans le domaine de " ERP/SAP et logistique " :

   a) Enregistrement des actifs dans SAP S/4HANA ;

   b) " Statistical Analysis System Enterprise Guide " 1 - Requêtes et rapports ;

   c) " Postgraduaat Enterprise Resource Planning " - Consultant fonctionnel ;

   d) EWM100: " Processes in SAP Extended Warehouse Management " ;

6. les formations suivantes dans le domaine : " Audit, fraude et gestion des risques " ;

   a) Certificat en Techniques de Gestion des Risques ;

   b) Masterclass " Internal Audit Essentials " ;

   c) Masterclass " Fraud Auditing " ;

7. les formations suivantes dans le domaine "  Gestion de projet, analyse métier et conduite du changement  " :

   a) Cours en Gestion des permis d'environnement à Bruxelles ;

   b) Certificat en Gestion de projet ;

   c) Certification en Conduite du Changement ;

   d) Analyste fonctionnel ;

   e) Certificat inter-universités en " User Experience Design and Research " (UX) ;

   f) Certificat d'université en Project and Change Management ;

   g) Formation et certification EXIN BCS " Business Analysis Foundation " ;

   h) Passation Des Marchés - Travaux, Fournitures et Consultants ;

8. les formations suivantes dans le domaine de " Immobilier, construction et facility management " :

   a) AutoCAD " Computer Aided Design " ;

   b) Revit ;

   c) Facility Management - Formation pratique ;

   d) " Postgraduaat " en Gestion immobilière ;

   e) " Postgraduaat " Construction Circulaire ;

9. la formation " Acheteur public " suivante dans le domaine " Marchés publics " ;

10. La formation Certificat inter-universités en Evaluation des Politiques Publiques dans le domaine " Droits de l'homme et politique gouvernementale " ;

11. les formations suivantes dans le domaine "  Leadership, coaching et développement des organisations " :

   a) " Intern Organisatiecoach " ;

   b) Masterclass " Start to Lead " ;

   c) Trajet de Développement pour Dirigeants ;

   d) Certificat d'université en Management Humain Innovant ;

   e) Vitruvius Diriger en Pleine Conscience ;

   f) " Global Learning and Development Leadership Diploma Program " ;

   g) Accréditation " Insights Discovery " ;

12. les formations suivantes dans le domaine "  Langues " :

   a) Français - 5ième année CVO Pro ;

   b) Français - Niveau 7 Vantage B2.1 ;

   a) " Black Belt " Excel ;

   c) " Postgraduaat " Utilisation et gestion des données dans l'administration publique ;
