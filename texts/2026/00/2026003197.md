**Article 1er.** La liste des métiers en pénurie pour les rôles linguistiques français et néerlandais est fixée pour l'année 2026 comme suit :

1. Expert en intégration et implémentation ;

2. Gestionnaire de base de données ;

3. Gestionnaire d'exploitation informatique ;

4. Responsable de département informatique ;

5. Analyste financier ;

6. Analyste-développeur TIC ;

7. Architecte IT;

8. Business Analyst ;

9. Architecte;

10. Archiviste ;

11. Assistant de direction ;

12. Formateur pour adultes ;

13. Ouvrier de voirie ;

14. Assistant social ;

15. Chauffeur permis C - CE ;

16. Chef de secteur des gardiens de parc ;

17. Chef de secteur des jardiniers ;

18. Comptable ;

19. Conducteur de chantier ;

20. Conseiller en prévention niveau 1 et 2 ;

21. Conseiller technique marché et réseaux ;

22. Contremaître ;

23. Contrôleur de travaux ;

24. Contrôleur de travaux à orientation espaces verts ;

25. Dessinateur technique ;

26. Electricien ;

27. Electromécanicien ;

28. Expert HVAC ;

29. Facilities manager ;

30. Gestionnaire de bâtiment ;

31. Gestionnaire de budget ;

32. Gestionnaire de marché public ;

33. Ingénieur ;

34. Inspecteur de conformité ;

35. Inspecteur du logement ;

36. Jardinier ;

37. Juriste ;

38. Manager de la mobilité ;

39. Manager Payroll ;

40. Plombier ;

41. Porte-parole ;

42. Technicien en froid et climatisation ;

43. Manager opérationnel.

**Art. 2.** Le présent arrêté produit ses effets au 1er janvier 2026.

**Art. 3.** Le ministre de la Fonction publique est chargé de l'exécution du présent arrêté.
