**Article 1er.** Les référentiels de validation des compétences relatifs au métier de Peintre décorateur sont approuvés comme suit :

_Table:_

| Métier  | Titres  | Code  |
| --- | --- | --- |
| Peintre décorateur  | UC1 : Réaliser manuellement des travaux de peinture sur différents supports  | PEINTDEC -1 |
|  | UC2 : Réaliser des travaux de peinture au pistolet  | PEINTDEC -2 |
|  | UC3 : Réaliser la pose d'un revêtement mural souple  | PEINTDEC -3 |
|  | UC4 : Réaliser la pose collée d'un revêtement de sol souple  | PEINTDEC -4  |




**Art. 2.** Le présent arrêté entre en vigueur le 27 avril 2017.

Le Membre du Collège ayant la Formation professionnelle dans ses attributions est chargé de l'exécution du présent arrêté.
