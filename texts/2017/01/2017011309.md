**Article 1er.** Les référentiels de validation des compétences relatifs au métier de Maçon-Maçonne sont approuvés comme suit :

_Table:_

| Métier  | Titres  | Code  |
| --- | --- | --- |
| Maçon/Maçonne  | UC1 : Poser des systèmes d'égouttage et de drainage périphérique  | MAC-1 |
|  | UC3 : Placer une isolation thermique sur un mur existant  | MAC-3 |
|  | UC4 : Réaliser une maçonnerie collée  | MAC-4 |
|  | UC6 : Réaliser une maçonnerie en blocs béton avec pose de poutrains/claveaux  | MAC-6 |
|  | UC7 : Réaliser une maçonnerie de parement au mortier  | MAC-7  |




**Art. 2.** Le présent arrêté entre en vigueur le 6 décembre 2016.
