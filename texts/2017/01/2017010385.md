**Article 1er.** Dans l'article 1er de l'arrêté ministériel du 20 juillet 2015 relatif aux zones de médecins généralistes, les modifications suivantes sont apportées :

1. le point 5°  est abrogé ;

2. dans le point 22°  est inséré le membre de phrase ", Desteldonk, Mendonk, Gentbrugge" ;

3. dans le point 38°  est inséré le membre de phrase ", Londerzeel " ;

4. le point 47°  est remplacé par ce qui suit :

" 47° Noord-Antwerpen : Merksem, Schoten, Antwerpen (2030), Berendrecht, Ekeren, Stabroek " ;

**Art. 2.** Le présent arrêté produit ses effets le 1er janvier 2017.
