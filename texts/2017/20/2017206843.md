**Article 1er.** Article unique. A partir du 1er janvier 2018, l'analyse des substances suivantes devient obligatoire pour les contrôles de surveillance des eaux souterraines requis pour les producteurs d'eau, conformément à l'article R.43bis et à l'annexe XI de la partie réglementaire du Livre II du Code de l'Environnement, contenant le Code de l'Eau :

- Code ESO 4618, desphenylchloridazon, alias MET-B;

- Code ESO 4497, chlorothalonil SA, alias VIS-01;

- Code ESO 4499, metazachlore ESA, alias BH479-8;

- Code ESO 4620, metolachlore ESA, alias CGA354743.
