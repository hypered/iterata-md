**Article 1er.** En application de l'article 85, § 5, alinéa 2, du décret du 15 juillet 1997 contenant le code flamand du Logement, la liste des communes est fixée comme suit :

1. Anvers ;

2. Baerle-Duc ;

3. Boom ;

4. Termonde ;

5. Dilsen-Stokkem ;

6. Duffel ;

7. Genk ;

8. Gand ;

9. Hamme ;

10. Courtrai ;

11. Kuurne ;

12. Lokeren ;

13. Maaseik ;

14. Maasmechelen ;

15. Malines ;

16. Menin ;

17. Messines ;

18. Olen ;

19. Schelle ;

20. Sint-Amands ;

21. Espierres-Helchin ;

22. Tamise ;

23. Furnes ;

24. Vilvorde ;

25. Wachtebeke ;

26. Wervik ;

27. Wezembeek-Oppem ;

28. Willebroek ;

29. Zelzate ;

30. Zwevegem ;

31. Zwijndrecht.

**Art. 2.** Le Ministre flamand compétent pour le logement est chargé de l'exécution du présent arrêté.
