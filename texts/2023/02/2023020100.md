**Article 1er.** Dans l'article 1er de l'arrêté du Gouvernement flamand du 15 juillet 2022 fixant la répartition des droits de vote entre les communes et les CPAS, situés dans la zone d'activité " Gent-Zuid " des sociétés de logement, l'alinéa 1er est remplacé par ce qui suit :

" Pour la société de logement qui est reconnue pour la zone d'activité " Gent-Zuid ", la répartition des droits de vote entre les communes et les CPAS situés dans cette zone d'activité est fixée comme suit :

1. commune d'Aalter : 13,46 % ;

2. commune de Deinze : 20,55 % ;

3. commune de De Pinte : 5,00 % ;

4. commune de Destelbergen : 6,09 % ;

5. commune de Gavere : 5,07 % ;

6. commune de Lievegem : 15,33 % ;

7. commune de Melle : 5,86 % ;

8. commune de Merelbeke : 10,32 % ;

9. commune de Nazareth : 5,42 % ;

10. commune de Sint-Martens-Latem : 5,00 % ;

11. commune de Zulte : 7,90 %. ".

**Art. 2.** Le ministre flamand compétent pour la Politique du logement est chargé de l'exécution du présent arrêté.
