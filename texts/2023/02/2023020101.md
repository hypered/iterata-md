**Article 1er.** Dans l'article 1er de l'arrêté du Gouvernement flamand du 15 juillet 2022 fixant la répartition des droits de vote entre les communes et les CPAS, situés dans la zone d'activité " Halle-Vilvoorde-Midden " des sociétés de logement, l'alinéa 1er est remplacé par ce qui suit :

" Pour la société de logement qui est reconnue pour la zone d'activité " Halle-Vilvoorde-Midden ", la répartition des droits de vote entre les communes et les CPAS situés dans cette zone d'activité est fixée comme suit :

1. commune d'Affligem : 4,59 % ;

2. commune d'Asse : 18,04 % ;

3. commune de Biévène : 0,76 % ;

4. commune de Dilbeek : 26,09 % ;

5. commune de Gammerages : 3,60 % ;

6. commune de Gooik : 3,81 % ;

7. commune de Herne : 2,18 % ;

8. commune de Lennik : 2,64 % ;

9. commune de Liedekerke : 5,46 % ;

10. commune de Merchtem : 6,76 % ;

11. commune d'Opwijk : 5,48 % ;

12. commune de Roosdaal : 5,21 % ;

13. commune de Ternat : 10,32 % ;

14. commune de Wemmel : 5,06 %. ".

**Art. 2.** Le ministre flamand compétent pour la Politique du logement est chargé de l'exécution du présent arrêté.
