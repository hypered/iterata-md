**Article 1er.** Dans l'article 1er de l'arrêté du Gouvernement flamand du 15 juillet 2022 fixant la répartition des droits de vote entre les communes et les CPAS, situés dans la zone d'activité " Vlaamse Ardennen-West " des sociétés de logement, l'alinéa 1er est remplacé par ce qui suit :

" Pour la société de logement qui est reconnue pour la zone d'activité " Vlaamse Ardennen-West ", la répartition des droits de vote entre les communes et les CPAS situés dans cette zone d'activité est fixée comme suit :

1. commune de Brakel : 8,05 % ;

2. commune de Horebeke : 0,48 % ;

3. commune de Kluisbergen : 5,57 % ;

4. commune de Kruisem : 9,30 % ;

5. commune de Lierde : 4,68 % ;

6. commune de Maarkedal : 3,12 % ;

7. commune d'Audenarde : 30,74 % ;

8. commune de Renaix : 30,02 % ;

9. commune de Wortegem-Petegem : 8,04 %. ".

**Art. 2.** Le ministre flamand compétent pour la Politique du logement est chargé de l'exécution du présent arrêté.
