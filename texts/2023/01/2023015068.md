**Article 1er.** Le coût moyen brut pondéré annuel d'un membre du personnel d'une Haute Ecole est, pour les groupes de fonctions suivants et pour l'année 2023, fixé comme suit :

   1) a) pour les maîtres-assistants nommés ou désignés dans l'enseignement supérieur de type court : 81.713 EUR ;

   b) pour les maîtres de formation pratique et les maîtres principaux de formation pratique : 72.131 EUR ;

   c) pour les maîtres-assistants nommés ou désignés dans l'enseignement supérieur de type long : 79.850 EUR ;

   2) a) pour les chargés de cours et les chefs de travaux : 99.883 EUR ;

   b) pour les professeurs et les chefs de bureau d'études : 118.487 EUR ;

   3) pour les directeurs et les directeurs-présidents : 108.623 EUR ;

   4) a) pour les membres du personnel administratif : 60.869 EUR ;

   b) pour les membres du personnel auxiliaire d'éducation : 70.669 EUR.

**Art. 2.** La Ministre de l'Enseignement supérieur est chargée de l'exécution du présent arrêté.

**Art. 3.** Le présent arrêté produit ses effets le 1er janvier 2023.
