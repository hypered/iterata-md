**Article 1er.** Dans l'article 1er de l'arrêté du Gouvernement flamand du 15 juillet 2022 fixant la répartition des droits de vote entre les communes et les CPAS, situés dans la zone d'activité " Midwest-West " des sociétés de logement, l'alinéa 1er est remplacé par ce qui suit :

" Pour la société de logement qui est reconnue pour la zone d'activité " Midwest-West ", la répartition des droits de vote entre les communes et les CPAS situés dans cette zone d'activité est fixée comme suit :

1. commune d'Ardooie : 3,87 % ;

2. commune de Hooglede : 3,68 % ;

3. commune d'Ingelmunster : 3,82 % ;

4. commune d'Izegem : 12,34 % ;

5. commune de Langemark-Poelkapelle : 3,90 % ;

6. commune de Ledegem : 4,35 % ;

7. commune de Lichtervelde : 3,55 % ;

8. commune de Moorslede : 4,97 % ;

9. commune de Poperinge : 8,69 % ;

10. commune de Roulers : 32,02 % ;

11. commune de Staden : 4,69 % ;

12. commune de Torhout : 6,43 % ;

13. commune de Vleteren : 2,38 % ;

14. commune de Zonnebeke : 5,31 %. ".

**Art. 2.** Le ministre flamand compétent pour la Politique du logement est chargé de l'exécution du présent arrêté.
