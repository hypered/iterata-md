**Article 1er.** Dans l'article 1er de l'arrêté du Gouvernement flamand du 15 juillet 2022 fixant la répartition des droits de vote entre les communes et les CPAS, situés dans la zone d'activité " Antwerpen-Midden/Noord " des sociétés de logement, l'alinéa 1er est remplacé par ce qui suit :

" Pour la société de logement qui est reconnue pour la zone d'activité " Antwerpen-Midden/Noord ", la répartition des droits de vote entre les communes et les CPAS situés dans cette zone d'activité est fixée comme suit :

1. commune de Brasschaat : 14,26 % ;

2. commune de Brecht : 11,21 % ;

3. commune d'Essen : 8,49 % ;

4. commune de Kalmthout : 4,18 % ;

5. commune de Kapellen : 5,90 % ;

6. commune de Malle : 8,40 % ;

7. commune de Ranst : 3,05 % ;

8. commune de Schilde : 3,79 % ;

9. commune de Schoten : 10,49 % ;

10. commune de Stabroek : 6,50 % ;

11. commune de Wijnegem : 2,02 % (arrondi) ;

12. commune de Wommelgem : 2,23 % ;

13. commune de Wuustwezel : 10,91 % ;

14. commune de Zandhoven : 3,06 % ;

15. commune de Zoersel : 5,51 %. ".

**Art. 2.** Le ministre flamand compétent pour la Politique du logement est chargé de l'exécution du présent arrêté.
