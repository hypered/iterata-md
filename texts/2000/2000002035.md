**Article 1er.** La rubrique III, 3 de l'annexe à l'arrêté royal du 22 décembre 1999 autorisant des administrations et autres services des ministères et des organismes d'intérêt public à engager des contractuels en vue de répondre à des besoins exceptionnels et temporaires en personnel est remplacée par la rubrique suivante :

```
  ---------------------------------------------------------------------------
    Commission permanente   |     | 4 | conseiller-adjoint | voir art. 3 AR |
    de Recours des Refugies |     |   |                    |                |
                            |     |   |                    |                |
    Gestion generale        |     | 6 | assistant          |      "         |
                            |     |   | administratif      |                |
                            |     |   |                    |                |
                            |     | 1 | commis             |      "         |
  ---------------------------------------------------------------------------
```



**Art. 2.** Le présent arrêté produit ses effets le 30 août 2000.

**Art. 3.** Notre Ministre de l'Intérieur est chargé de l'exécution du présent arrêté.

Donné à Bruxelles, le 7 janvier 2001.
